### Summary of the revisions made

| Change | What was altered & why |
|--------|------------------------|
| **Merged financial abuse into coercive control** | The Delphi panel showed most experts treat money‑restriction as a tool of broader control. All financial‑related items (34, 35, 48) are now in **Factor 1**. |
| **Integrated jealousy / possessiveness** | Items that convey extreme jealousy (e.g., “vorrebbe che passassimo la maggior parte del tempo insieme”) were placed in **Coercive Control**, reflecting the majority view that jealousy is a surveillance tactic. |
| **Moved mood‑oscillation item (1) to Emotional Abuse** | Although mood volatility can create intimidation, it primarily functions as a de‑valuing, destabilising verbal pattern, so it fits best with **Emotional Abuse & Degradation**. |
| **Placed non‑consensual image sharing (38) under Threats** | The image‑sharing behaviour is used to threaten the victim, aligning with the “Threats & Intimidation” theme identified by the Delphi synthesis. |
| **Created a distinct Fear/Hyper‑vigilance factor** | Item 30 explicitly describes hiding behaviour out of fear; the Delphi panel highlighted “Fear / hyper‑vigilance” as a separate theme, so a dedicated factor was added. |
| **Collapsed Reputation/Image Abuse into other factors** | Items that involve public shaming (23, 40) were already covered by **Emotional Abuse** or **Isolation**, making a separate “Reputation” factor redundant. |
| **Removed the separate “Gas‑lighting” vs. “Emotional Abuse” debate** | Following the synthesis, **Gas‑lighting** is retained as its own factor (Factor 3) because several experts argued it represents a distinct cognitive‑distortion process. |
| **Omitted a reverse‑coded healthy‑relationship factor** | The original questionnaire does not contain the item “Sento di potergli parlare di qualsiasi cosa”; therefore the factor is listed only as a placeholder, with a note that no item is available. |
| **Ensured non‑overlap** | Every original item appears **once and only once** in the table, satisfying the “no overlap” requirement. |
| **Reduced the total number of factors from nine to eight** | By merging financial abuse, jealousy, and some reputation items, and by adding the fear dimension, we achieve a parsimonious yet comprehensive structure that mirrors the consensus seven themes plus the reverse‑coded check. |

### How to use this model

1. **Scoring** – Sum the Likert‑type responses for the items belonging to each factor to obtain a sub‑scale score (higher = greater severity).  
2. **Validity check** – If a reverse‑coded healthy‑relationship item is later added, its score can be used to flag careless or socially desirable responding.  
3. **Clinical mapping** –  
   * Factor 1 → ICD‑11 “Coercive control” / DSM‑5‑TR “Intimate‑partner relational problem”.  
   * Factor 2 → DSM‑5‑TR “Emotional abuse”.  
   * Factor 3 → DSM‑5‑TR “Psychological maltreatment (gaslighting)”.  
   * Factor 4 → DSM‑5‑TR “Intimidation / Threats”.  
   * Factor 5 → DSM‑5‑TR “Physical and sexual violence”.  
   * Factor 6 → DSM‑5‑TR “Social isolation”.  
   * Factor 7 → DSM‑5‑TR “Fear and hyper‑vigilance”.  

Researchers can run an **exploratory factor analysis (EFA)** on the full item set to test whether the data empirically support this eight‑factor solution, and then confirm the structure with a **confirmatory factor analysis (CFA)** in an independent sample. The model is deliberately aligned with the most widely‑accepted theoretical frameworks while respecting the nuances highlighted by the Delphi experts.
